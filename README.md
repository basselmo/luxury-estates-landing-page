# 🏠 Luxury Estates Landing Page

A modern, responsive, and high-conversion landing page designed for real estate agencies and property developers. 

## 🚀 Live Demo
You can view the live website here: [**PUT_YOUR_GITHUB_PAGES_LINK_HERE**]

## ✨ Features
* **Fully Responsive:** Works perfectly on Desktops, Tablets, and Mobile phones.
* **Modern UI/UX:** Clean design with smooth hover effects and professional typography.
* **Dynamic Backgrounds:** Feature cards with high-quality background images.
* **Functional Contact Form:** Integrated with **Formspree** to receive client inquiries directly via email.
* **Testimonials Section:** Social proof with client icons and ratings.
* **Project Gallery:** Showcase of premium properties.

## 🛠️ Technologies Used
* **HTML5:** Semantic structure.
* **CSS3:** Flexbox, Grid, and Media Queries for responsiveness.
* **Google Fonts:** Poppins for a premium look.
* **Formspree:** For backend-less form handling.
